About

This theme is based on Twitter Bootstrap 3 grid system, so there are some additional files included.

index.html is the core file of the template.

php/process.php this file makes the registration and subscribe forms work

php/subscribers.txt here will apear a list of all people that subscribed via subscribe form

/css/ folder

In this folder you can find all .css files of this theme. If you would like to change some styles edit only style.css

/js/ folder

Here you can find all .js files. If you want to change some javascript files, you can only change main.js

/images/ folder

Here are located all images used in this theme

HTML Structure

In this template you can move different sections ( Section example: Speakers, F.A.Q, Schedule ) up and down, in the way it fits your needs.

A section look's like this:

<!--===============================-->
<!--== Section Title ==============-->
<!--===============================-->
    Section Content
<!--==========-->

and it can be moved anywhere inside of this container:

<div class="content">
    <div class="container box">
        Sections Here
    </div>
</div>

Do not place a section inside another section!

div.fancy-select ul.options li.selected, div.fancy-select ul.options li.selected.hover {
    background-color: #2ecc71; /* <- Paste here your prefered color */
}

a, .accordion-heading span, .speaker-contacts > .contact,
.table-features li.fa-check-circle, .owl-btn.prev,
.owl-btn.next, div.fancy-select ul.options li.hover {
    color: #2ecc71; /* <- Paste here your prefered color */
}

.speaker:hover .speaker-photo, .speaker-contacts > .contact,
.social-icons li a:hover, .gotop:hover, .speaker-contacts > .contact:hover {
    border-color: #2ecc71; /* <- Paste here your prefered color */
}

Replace "#2ecc71" with your prefered color.

Menu

The order of the links in the navigation bar MUST match the order of the sections as laid out on the page or unexpected behavior will occur.

Each menu link should have a href targeting coresponding section id in the DOM.

Example bellow: ( href="#faq" targeting id="faq" )

<ul class="nav navbar-nav">
	<li><a href="#faq">FAQ</a></li>
	...
</ul>

<!--===============================-->
<!--== FAQ ========================-->
<!--===============================-->
<section id="faq" class="row">
	...
</section>
<!--==========-->

Logo

You can change logo in 2 ways.

Chose an icon that you would like to use as your logo from FontAwesome and replace the one used by default "fa-map-marker", and then replace the default site name with yours.

<!--== Logo ==-->
<span class="navbar-brand logo">
    <span class="fa fa-map-marker"></span>Event
</span>

And the second way, is by using an image:

<span class="navbar-brand logo">
    <img src="images/yourlogo.png" alt="Logo"/>
</span>

where src="images/yourlogo.png" - path to your image.

Sliders

Adding and removing slides is simple. In this template we have 2 sliders, the big one in the header, and a small one in content, but they both work the same.

A slider HTML looks like this:

<ul class="slides">
    <li><img src="images/demo/slide-01.jpg" alt="Slide 1"/></li>
    ...
</ul>

src="images/demo/slide-01.jpg" - path to your image

alt="Slide 1" - slide title

Tabs

Working with tabs is pretty simple as well.

This is how tabs HTML look, remember to activate each tab element and content container. Tab should have a href targeting a container id in the DOM.

Example bellow: ( href="#tab1" targeting id="tab1" )

<div class="content-tabs">

    <!-- Tab Navigation -->
    <ul class="nav nav-tabs">
      <li class="active"><a href="#tab1" data-toggle="tab">First Tab Title</a></li>
      <li><a href="#tab2" data-toggle="tab">Second Tab Title</a></li>
      ...
    </ul>

    <!-- Tab Content -->
    <div class="tab-content">
        <div class="tab-pane active" id="tab1">
            First Tab Content
        </div>
        <div class="tab-pane" id="tab2">
            Second Tab Content
        </div>
        ...
    </div>

</div>

 

Carousels

If you want to add/remove some images to carousel, all you have to do is edit this lines:

<div id="carousel" class="owl-carousel">
    <div class="item"><img src="images/demo/partners/logo-01.png" alt="Logo 1"></div>
    ...
</div>

each image:

<div class="item"><img src="images/demo/partners/logo-01.png" alt="Logo 1"></div>

src="images/demo/partners/logo-01.png" - path to your image

alt="Logo 1" - image title

 

You can add as many images as you need.

Accordion

Accordion's are that nice collapsing panels, used in Schedule and F.A.Q Sections.

This is how accordion HTML looks like:

<ul class="accordion-group">
    <li class="accordion">
        <a class="accordion-heading">
            <time>10:30</time>
            <h4 class="accordion-title">Fugit, officia laudantium repellat</h4>
            <span class="fa fa-plus-circle pull-right"></span>
        </a>
        <div class="accordion-body">
            Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ex, ratione nostrum atque nobis debitis doloremque. Fugit, officia laudantium repellat assumenda eligendi vel sed dolore delectus. Vitae sequi cum doloribus veniam.
        </div>
    </li>
    ...
</ul>

each panel:

    <li class="accordion">
        <a class="accordion-heading">
            <time>10:30</time>
            <h4 class="accordion-title">Fugit, officia laudantium repellat</h4>
            <span class="fa fa-plus-circle pull-right"></span>
        </a>
        <div class="accordion-body">
            Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ex, ratione nostrum atque nobis debitis doloremque. Fugit, officia laudantium repellat assumenda eligendi vel sed dolore delectus. Vitae sequi cum doloribus veniam.
        </div>
    </li>

<time> - is only needed for schedule accordions, they are missing in F.A.Q Section.

 

You can add as many accordion's as you need.

Register

Our registration form, send's you an mail each time a user is registered, so all you have to do is open register.php

and edit this line:

// Your Email Address
$youremail = "Your Email Adress Here";

 

( Example: $youremail = "example@example.com"; )
